# NexaPay - University Digital Wallet

This is a high-fidelity mobile web prototype for NexaPay, designed with a premium dark theme and smooth animations.

## Prerequisites

You need **Node.js** installed on your computer to run this application.

## How to Run

1.  Open a terminal in this folder.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the link shown in the terminal (usually `http://localhost:5173`) in your browser.

## Features

-   **Mobile-First Design**: The app is constrained to a mobile viewport size on desktop for a realistic app simulation.
-   **Dark Mode**: Premium dark aesthetic with gradients.
-   **Animations**: Smooth page transitions and micro-interactions using Framer Motion.
-   **Modules**:
    -   **Login**: Dummy login with validation.
    -   **Dashboard**: Overview of balance and transactions.
    -   **Savings**: Visual goal tracking.
    -   **Transfer**: Simulated money transfer interface.
    -   **Discounts**: Student offers with coupon codes.

## Note

This is a frontend prototype. Data is static and resets on reload. No actual money is moved.
